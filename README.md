# AI Bioactivity Predictor (NR-AR)

## 🚀 Overview
This project predicts whether a chemical compound shows activity against the Androgen Receptor (AR) using machine learning.

It serves as an early-stage screening tool to identify potentially bioactive molecules before expensive lab testing.

---

## 🧠 Features
- Input: SMILES string (chemical structure)
- Predicts: AR Bioactivity (Active / Inactive)
- Uses:
  - Morgan Fingerprints (2048 bits)
  - Machine Learning Model (Ensemble)
  - SHAP Explainability
- Displays:
  - Molecular properties
  - Structure visualization
  - Confidence score

---

## 🛠️ Tech Stack
- Python
- Streamlit
- RDKit
- Scikit-learn
- SHAP
